Deze repo host uitsluitend de **installers en updatermetadata** van
[Omnissiah](https://robinvliet.github.io/omnissiah-releases/), een persoonlijke,
self-hosted AI-workspace. De broncode is privé.

- **Downloadpagina:** https://robinvliet.github.io/omnissiah-releases/
- **Alle releases:** zie het [Releases-tabblad](../../releases)

De desktop-app controleert bij het opstarten op nieuwe versies in deze repo
(niets wordt zonder bevestiging gedownload of geïnstalleerd).
